Умный бот-прогноз погоды с советами по одежде, геолокацией и уведомлениями!

✨ Что умеет бот

🌍 Погода по гео — текущая погода по вашей локации

📍 Поиск городов — погода в любом городе мира

👗 Что одеть — персональные советы по "ощущаемой" температуре

📅 Прогноз — на 5 дней вперед

🕐 По часам — детальный прогноз на 12 часов

🥇 Топ городов — сравнение температуры до 3 городов

⏰ Подписка — автоуведомления каждые 30 минут

⭐ Избранное — сохранение любимых городов

📊 Статистика — ваши запросы и города




Пример: 🌤️ Москва

🌡 -5°C ❄️

💧 78% • 💨 3м/с

📉 1008гПа

❄️ **ЗИМА** (-3-4°C)

👚 Термобельё + флис

🧥 Пуховик

🧣 Шарф + шапка

🧤 Вязаные перчатки


📅 *Прогноз:

❄️ Сегодня: -4°/-7°

🌤️ Завтра: -2°/-5°

Пн: 0°/-3°


Open-Meteo API (бесплатно, без ключей)

Geocoding API (поиск городов)

pyTelegramBotAPI

Threading (уведомления)

💾 In-memory storage



Как установить 
# 1. Клонируем
git clone https://github.com/yourusername/weatherbot.git
cd weatherbot

# 2. Устанавливаем
pip install -r requirements.txt

# 3. Создаем бота @BotFather → получаем TOKEN
# 4. Запускаем
python bot.py
